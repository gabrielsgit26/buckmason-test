# Product Listing Page

Responsive product listing UI inspired by the clean, minimal Buck Mason aesthetic ([buckmason.com](https://www.buckmason.com/)).

## Features

- Product grid (8 hardcoded items)
- Category, color, and max price filters
- Product detail modal
- Shopping cart drawer with add/remove/update quantity
- Responsive layout for mobile and desktop
- Tailwind CSS based minimal styling and transitions

## Tech Stack

- React (Vite)
- Tailwind CSS
- No backend / no database

## Run

```bash
npm install
npm run dev
```
